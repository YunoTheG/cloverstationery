About Us

Clovers Stationery is an online stationery shop project inspired by Nepal's creativity and simplicity. This website showcases fundamental web design skills using only HTML and CSS, demonstrating clean layout structure, responsive design, and interactive visual effects without any JavaScript.

The project represents a blend of traditional Nepali aesthetics with modern web design principles, featuring a calming color palette inspired by Nepal's natural beauty and study culture.

Every element on this page—from the smooth animations to the hover effects—is created purely with CSS, showing the power of modern web styling techniques.
